Key Features

Advanced Field Types

Go beyond basic text and multiple-choice with over 25 field types including Star Ratings, Range Sliders, Signatures, File Uploads, Geolocation, and more.

DataMate Integration

Unique ability to trigger custom data processing with the "DataMate Record" option, copying and saving form data across multiple sheets with complex logic.

Flexible Submission Options

Choose between Single Cell or Table Row submission types, allowing precise control over how data is stored in your spreadsheet.

Dynamic Dropdowns

Populate dropdowns from spreadsheet ranges or custom lists, with support for formulas to keep options current.

Conditional Logic

Create smart forms with fields that appear based on user responses, enhancing user experience and data relevance.

Custom Styling

Beautiful, modern design with customizable containers, headers, and footers for a professional look tailored to your brand.

File Management

Upload files directly to Google Drive with automatic link generation, maintaining organization and accessibility.
