How to Seamlessly Integrate Data with this app

Welcome to our tutorial on harnessing the power of DataMate to streamline your business management using spreadsheets. Follow these steps to make your workflow more efficient:

Step 1: Adding Data

If you use other spreadsheets (Excel Worksheets, Sheets, even web pages) you can easily add them to your file:

  • Enter your data or locate the file you want to integrate, for example, an Invoice form, Work Order, Timesheet ect...ect...ect (Anything).
  • Merge all cells that contain data or text.
  • Copy and paste data or form from the file directly into cells A3:Q48 of the Input sheet of the DataMate App begining at cell A3. Use the 'Paste Column Width' function if available.
  • Copy and paste data from the Input sheet directly into (cells A3:Q48) of the View/Print sheet begining at cell A3. Use the 'Paste Column Width' function if available. This step is required to provide formating of the sheet. The data can be deleted. Selet the DataMate App and click Refresh View.

Step 2: Setup the Log table

  • On the Input sheet select the 'Log 1' cell then press '=' and select the cell of your data that contains the label of the unique ID of your data (ie.Invioce No., PO No., Report No. ect.) press 'Enter'.
  • Select the cell below Log 1 cell and press '=' then select the cell of your data that contains the data of the unique ID. Press 'Enter'.
  • Repeat this step for up to 12 table fields.

Step 3: Save, View, Print and Update your data.

  • Select the DataMate App and press Save Data. The screen will change to the View/Print sheet when complete. Save up to 10,000 records per Data sheet!
  • Select the record from the green shaded dropdown box on the View/Print sheet.
  • View the Log and Update sheets.
  • To update the record, enter information for the desired record on the Update sheet (columns B, C or D).
  • To print, select the cells on the View/Print sheet you would like to print. Select 'File' then 'Print'. In 'Page setup' select 'Print' - 'Current selection'.

    • Change the entries on the Input sheet and save to create another record.

Congratulations! You've successfully integrated your data into your management workflow using the DataMate App. Enjoy the benefits of a more organized and efficient management experience.
