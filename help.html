DataMate Tutorial

Welcome to DataMate! This comprehensive guide covers how to use DataMate for data management, form building, and contact management within Google Sheets. Current date: April 08, 2025.

DataMate is a Google Sheets add-on designed to simplify data management, form creation, and contact organization. Key capabilities include:

  • Automated Data Entry: Organize data into structured sheets.
  • Record Management: Save, view, update, and print records easily.
  • Customizable Forms: Build advanced forms with over 25 field types, including dynamic tables for interactive data entry.
  • Contact Integration: Import and manage contacts from Gmail or CSV files.
  • Dynamic Logs: Track up to 12 log fields per record.

Installation

Install DataMate from the Google Workspace Marketplace. Open a Google Sheet, and look for the "Form Tools" menu.

Initialize Your Spreadsheet

  • New Spreadsheet: Open a new Google Sheet and select "Form Tools > Preview Form" to create the FormSetup sheet.
  • Existing Spreadsheet: Ensure a FormSetup sheet exists or run "Preview Form" to generate it.

Adding Data

Use forms created in FormSetup to input data, which can target multiple sheets and cells as defined in columns B:G.

Managing Records

  • Save: Submit a form to store data in target sheets (e.g., Responses, Input).
  • Custom Actions: Define "On Submit Functions" in FormSetup!A3 (e.g., save, copyInput1) to execute after submission.

Configure Settings

In the FormSetup sheet:

  • A3: On Submit Functions (e.g., "save, copyInput1, newContactit") - Comma-separated list of functions to run after form submission.

Note: The form name is set via a Header field in A9:I.

Add Fields

Edit A9:I in FormSetup:

  • A: Field Name
  • B: Target Sheet 1
  • C: Target Cell/Column 1 (e.g., A1 for single cell, A for table column)
  • D: Target Sheet 2 (optional)
  • E: Target Cell/Column 2 (optional)
  • F: Target Sheet 3 (optional)
  • G: Target Cell/Column 3 (optional)
  • H: Field Type
  • I: Options (e.g., dropdown values, formulas, or styling)

Supported Field Types

All supported field types with examples:

Text

A single-line text input.

  • I: Leave blank.
  • Example: ["Name", "Responses", "A", "", "", "", "", "Text", ""]

Dropdown

A dropdown menu with selectable options.

  • I: Comma-separated options or sheet range (e.g., "=Sheet1!A:A").
  • Example: ["Choice", "Responses", "B", "", "", "", "", "Dropdown", "Red,Blue,Green"]

MultiSelect

A dropdown allowing multiple selections.

  • I: Comma-separated options or sheet range.
  • Example: ["Interests", "Responses", "C", "", "", "", "", "MultiSelect", "Tech,Art,Sports"]

Date

A date picker input.

  • I: Leave blank.
  • Example: ["Event Date", "Responses", "D", "", "", "", "", "Date", ""]

Time

A time picker input.

  • I: Leave blank.
  • Example: ["Event Time", "Responses", "E", "", "", "", "", "Time", ""]

Number

A numeric input field.

  • I: Leave blank.
  • Example: ["Quantity", "Responses", "F", "", "", "", "", "Number", ""]

Checkbox

A single checkbox (true/false).

  • I: Leave blank.
  • Example: ["Urgent", "Responses", "G", "", "", "", "", "Checkbox", ""]

Radio

Radio buttons for single-choice selection.

  • I: Comma-separated options.
  • Example: ["Priority", "Responses", "H", "", "", "", "", "Radio", "Low,Medium,High"]

Textarea

A multi-line text input.

  • I: Leave blank.
  • Example: ["Comments", "Responses", "I", "", "", "", "", "Textarea", ""]

Email

An email input with validation.

  • I: Leave blank.
  • Example: ["Email", "Responses", "J", "", "", "", "", "Email", ""]

StarRating

A 5-star rating input.

  • I: Leave blank (fixed 1-5 scale).
  • Example: ["Rating", "Responses", "K", "", "", "", "", "StarRating", ""]

RangeSlider

A slider for selecting a value within a range.

  • I: "min,max,step" (e.g., "0,100,5").
  • Example: ["Satisfaction", "Responses", "L", "", "", "", "", "RangeSlider", "0,100,5"]

FileUpload

Allows file uploads (max 6MB).

  • I: Leave blank.
  • Behavior: Stores link in Google Drive.
  • Example: ["Attachment", "Responses", "M", "", "", "", "", "FileUpload", ""]

Conditional

A field shown based on another field's value.

  • I: "FieldName=Value" (e.g., "Urgent=Yes").
  • Example: ["Reason", "Responses", "N", "", "", "", "", "Conditional", "Urgent=true"]

Calculated

A read-only calculated field.

  • I: "=Formula" (e.g., "=Quantity*10").
  • Example: ["Total", "Responses", "O", "", "", "", "", "Calculated", "=Quantity*10"]

Signature

A canvas for drawing a signature.

  • I: Leave blank.
  • Behavior: Saves as PNG link in Google Drive.
  • Example: ["Signature", "Responses", "P", "", "", "", "", "Signature", ""]

Geolocation

Captures latitude and longitude.

  • I: Leave blank.
  • Behavior: User clicks to share location.
  • Example: ["Location", "Responses", "Q", "", "", "", "", "Geolocation", ""]

ProgressBar

Displays a static progress bar.

  • I: Number (0-100).
  • Example: ["Progress", "Responses", "R", "", "", "", "", "ProgressBar", "75"]

Captcha

A simple verification (fixed "3 + 5").

  • I: Leave blank.
  • Behavior: User enters "8".
  • Example: ["Verification", "Responses", "S", "", "", "", "", "Captcha", ""]

Image

Displays a static image.

  • I: Image URL (e.g., "https://via.placeholder.com/150").
  • Example: ["Product Image", "Responses", "T", "", "", "", "", "Image", "https://via.placeholder.com/150"]

Video

Displays a playable video.

  • I: Video URL (e.g., YouTube or direct link).
  • Example: ["Product Video", "Responses", "U", "", "", "", "", "Video", "https://www.youtube.com/watch?v=dQw4w9WgXcQ"]

ImageLink

Allows input of an image URL.

  • I: Leave blank.
  • Example: ["Upload Image URL", "Responses", "V", "", "", "", "", "ImageLink", ""]

VideoLink

Allows input of a video URL.

  • I: Leave blank.
  • Example: ["Upload Video URL", "Responses", "W", "", "", "", "", "VideoLink", ""]

StaticText

Displays static text.

  • I: Text to display.
  • Example: ["Welcome", "", "", "", "", "", "", "StaticText", "Please fill out this form."]

Table

Displays a static table from a sheet range.

  • I: Range (e.g., "Sheet2!A1:C5").
  • Example: ["Sales Data", "", "", "", "", "", "", "Table", "Sheet2!A1:C5"]

Container

Groups fields with custom styling.

  • I: Inline CSS or blank.
  • Example: ["Section 1", "", "", "", "", "", "", "Container", "background: #f0f0f0;"]

Header

A styled header section.

  • I: Text to display (also sets form name).
  • Example: ["Form Header", "Responses", "A", "", "", "", "", "Header", "Welcome"]

Footer

A styled footer section.

  • I: Text to display.
  • Example: ["Form Footer", "Responses", "B", "", "", "", "", "Footer", "Thank You"]

Preview & Deploy

Use "Form Tools > Preview Form" to test the form. Deploy via Apps Script: "Deploy > New Deployment > Web App."

Collect Data

Share the web app URL. Data saves to target sheets and cells as defined in B:G. On-submit functions in A3 execute post-submission.
