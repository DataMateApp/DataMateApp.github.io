DataMate Tutorial

Welcome to DataMate! This comprehensive guide covers how to use DataMate for data management, form building, and contact management within Google Sheets. Current date: April 06, 2025.

DataMate is a Google Sheets add-on designed to simplify data management, form creation, and contact organization. Key capabilities include:

  • Automated Data Entry: Organize data into structured sheets.
  • Record Management: Save, view, update, and print records easily.
  • Customizable Forms: Build advanced forms with over 25 field types, including dynamic tables for interactive data entry.
  • Contact Integration: Import and manage contacts from Gmail or CSV files.
  • Dynamic Logs: Track up to 12 log fields per record.

Installation

Install DataMate from the Google Workspace Marketplace. Open a Google Sheet, and look for the "DataMate" menu.

Initialize Your Spreadsheet

  • New Spreadsheet: Open a new Google Sheet, enter data in A3:Q48 on Sheet1, and select "DataMate > New Dataset" to set up specialized sheets (Input, Data, View_Print, Update, Log).
  • Existing Spreadsheet: Ensure Sheet1 has data in A3:Q48, then run "New Dataset."

Adding Data

Enter data in Input sheet cells A3:Q48. Merge cells containing text for proper formatting.

Setting Up Logs

In the Input sheet:

  • For each log (e.g., Log 1): Type =, select the label cell (e.g., "Invoice Number"), and press Enter.
  • Below each log label, type =, select the data cell, and press Enter. Up to 12 logs are supported.

Managing Records

  • Save: Click "DataMate > Save Record" to store data in the Data sheet and view it in View_Print.
  • View: Use the dropdown in View_Print!B2 to select records.
  • Update: Edit records in the Update sheet; changes sync to Data.
  • Print: In View_Print, go to "File > Print > Current Selection."
  • Reset: Use "Reset Input" or "Reset View/Print" to refresh sheets from Sheet1.

Configure Settings

In the FormSetup sheet:

  • B3: Form Name (e.g., "Customer Survey")
  • B4: Target Sheet (e.g., "Responses")
  • B5: Submission Type (SingleCell or TableRow)

Add Fields

Edit A9:E in FormSetup:

  • A: Field Name
  • B: Target Sheet
  • C: Target Cell/Column
  • D: Field Type
  • E: Options (e.g., dropdowns, formulas, or table configurations)

Supported Field Types

All supported field types with examples:

Text

A single-line text input.

  • E: Leave blank.
  • Example: ["Name", "Responses", "A", "Text", ""]

Dropdown

A dropdown menu with selectable options.

  • E: Comma-separated options or sheet range (e.g., "=Sheet1!A:A").
  • Example: ["Choice", "Responses", "B", "Dropdown", "Red,Blue,Green"]

MultiSelect

A dropdown allowing multiple selections.

  • E: Comma-separated options or sheet range.
  • Example: ["Interests", "Responses", "C", "MultiSelect", "Tech,Art,Sports"]

Date

A date picker input.

  • E: Leave blank.
  • Example: ["Event Date", "Responses", "D", "Date", ""]

Time

A time picker input.

  • E: Leave blank.
  • Example: ["Event Time", "Responses", "E", "Time", ""]

Number

A numeric input field.

  • E: Leave blank.
  • Example: ["Quantity", "Responses", "F", "Number", ""]

Checkbox

A single checkbox (true/false).

  • E: Leave blank.
  • Example: ["Urgent", "Responses", "G", "Checkbox", ""]

Radio

Radio buttons for single-choice selection.

  • E: Comma-separated options.
  • Example: ["Priority", "Responses", "H", "Radio", "Low,Medium,High"]

Textarea

A multi-line text input.

  • E: Leave blank.
  • Example: ["Comments", "Responses", "I", "Textarea", ""]

Email

An email input with validation.

  • E: Leave blank.
  • Example: ["Email", "Responses", "J", "Email", ""]

StarRating

A 5-star rating input.

  • E: Leave blank (fixed 1-5 scale).
  • Example: ["Rating", "Responses", "K", "StarRating", ""]

RangeSlider

A slider for selecting a value within a range.

  • E: "min,max,step" (e.g., "0,100,5").
  • Example: ["Satisfaction", "Responses", "L", "RangeSlider", "0,100,5"]

FileUpload

Allows file uploads (max 6MB).

  • E: Leave blank.
  • Behavior: Stores link in Google Drive.
  • Example: ["Attachment", "Responses", "M", "FileUpload", ""]

Conditional

A field shown based on another field's value.

  • E: "FieldName=Value" (e.g., "Urgent=Yes").
  • Example: ["Reason", "Responses", "N", "Conditional", "Urgent=Yes"]

Calculated

A read-only calculated field.

  • E: "FieldName=Formula" (e.g., "Total=Quantity*10").
  • Example: ["Total", "Responses", "O", "Calculated", "Total=Quantity*10"]

Signature

A canvas for drawing a signature.

  • E: Leave blank.
  • Behavior: Saves as PNG link in Google Drive.
  • Example: ["Signature", "Responses", "P", "Signature", ""]

Geolocation

Captures latitude and longitude.

  • E: Leave blank.
  • Behavior: User clicks to share location.
  • Example: ["Location", "Responses", "Q", "Geolocation", ""]

ProgressBar

Displays a static progress bar.

  • E: Number (0-100).
  • Example: ["Progress", "Responses", "R", "ProgressBar", "75"]

Captcha

A simple verification (fixed "3 + 5").

  • E: Leave blank.
  • Behavior: User enters "8".
  • Example: ["Verification", "Responses", "S", "Captcha", ""]

Image

Displays a static image.

  • E: Google Drive link (e.g., "https://drive.google.com/uc?id=FILE_ID").
  • Example: ["Product Image", "Responses", "T", "Image", "https://drive.google.com/uc?id=1a2b3c"]

Video

Displays a playable video.

  • E: Direct video URL.
  • Example: ["Product Video", "Responses", "U", "Video", "https://www.w3schools.com/html/mov_bbb.mp4"]

ImageLink

Allows input of an image URL.

  • E: Leave blank.
  • Example: ["Upload Image URL", "Responses", "V", "ImageLink", ""]

VideoLink

Allows input of a video URL.

  • E: Leave blank.
  • Example: ["Upload Video URL", "Responses", "W", "VideoLink", ""]

StaticText

Displays static text.

  • E: Text to display.
  • Example: ["Welcome", "Responses", "X", "StaticText", "Please fill out this form."]

Table

Displays a static table from a sheet range.

  • E: Range (e.g., "Sheet2!A1:C5").
  • Example: ["Sales Data", "Responses", "Y", "Table", "Sheet2!A1:C5"]

DynamicTable

An interactive table with dropdowns, price lookups, quantity inputs, and real-time totals (subtotal, tax, total).

  • E: "=SheetName!Column:Column,PriceColumn:Column" (e.g., "=Inventory!A:A,PriceColumn:C").
  • Behavior: Users add/remove rows; data serializes to a single cell as JSON.
  • Tax Rate: Set in Sheet1!C33 (e.g., 0.08 for 8%).
  • Example: ["Order Items", "Responses", "Z", "DynamicTable", "=Inventory!A:A,PriceColumn:C"]

Container

Groups fields with custom styling.

  • E: Inline CSS or blank.
  • Example: ["Section 1", "Responses", "AA", "Container", "background: #f0f0f0;"]

Header

A styled header section.

  • E: Text to display.
  • Example: ["Form Header", "Responses", "AB", "Header", "Welcome"]

Footer

A styled footer section.

  • E: Text to display.
  • Example: ["Form Footer", "Responses", "AC", "Footer", "© 2025"]

Preview & Deploy

Use "DataMate > FormBuilder > Preview Form". Deploy via Apps Script: "Deploy > New Deployment > Web App."

Collect Data

Share the URL. Data saves to the target sheet based on submission type. Dynamic table data is serialized as JSON in a single cell for TableRow submissions.

Setup Contact Sheets

Select "DataMate > AddressBlock > Add Contact Sheets" to create contacts, Address, and NewContact sheets.

Import Contacts

Use "Import Gmail™ Contacts" to upload a CSV file from Gmail or Outlook into contacts.

Watch the AddressBlock Video

Add New Contacts

Enter details in NewContact and save via "AddressBlock > Save New Contact."

Generate Address Blocks

In Address, set F1 to a cell reference, then use "AddressBlock > Address Block Name" or "Company" to populate contact data.

Tips

  • Use NewContact for data integrity.
  • Edit contacts directly for updates.
