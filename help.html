DataMate App Quick Start Guide

Welcome to the DataMate App! This guide will help you quickly get started with the basic functions of integrating and managing your data using spreadsheets.

Step 1: Adding Data

Follow these steps to add data to your DataMate file:

  1. Open your existing spreadsheet file (Excel™, Google Sheets™, CSV anything).
  2. Copy the data or form you want to integrate.
  3. Paste the data into cells A3:Q48 on the Input sheet of the DataMate App. Use the 'Paste Column Width' function if available.
  4. Repeat the paste operation on the View/Print sheet, starting at cell A3. This ensures proper formatting.
  5. Select the DataMate App and click Refresh View.

Step 2: Setup the Log Table

Set up the log table to keep track of your data records:

  1. On the Input sheet, select the cell labeled Log 1, then press = and select the cell containing the unique ID label (e.g., Invoice No.). Press Enter.
  2. Select the cell below Log 1, press =, and select the cell containing the unique ID data. Press Enter.
  3. Repeat this process for up to 12 fields as needed.

Step 3: Save, View, Print, and Update Your Data

Manage your data with the following actions:

  1. To save data, select the DataMate App and press Save Data. This switches to the View/Print sheet.
  2. To view a record, select it from the green dropdown box on the View/Print sheet.
  3. To update a record, enter the new information on the Update sheet (columns B, C, or D).
  4. To print, select the desired cells on the View/Print sheet, then go to File > Print, and choose Print Current Selection in the page setup.

Adding Contacts

Integrate your contacts into the DataMate App:

  1. Export your Gmail™ contacts to an Outlook™ CSV file.
  2. Open the CSV file, select all contacts (Ctrl+A), and copy them (Ctrl+C).
  3. Paste them (Ctrl+V) into the green shaded cell B1 on the Contacts sheet.
  4. Use the Name dropdown box on the Address sheet to copy and paste into the desired cells on your form or spreadsheet.
  5. Select the target cell on the Address sheet and enter the corresponding cell reference. Use 'Target formulas' from the menu to complete the integration. Also see AddressBlock Help

Congratulations! You've successfully started using the DataMate App. Enjoy the benefits of a more organized and efficient management experience.
