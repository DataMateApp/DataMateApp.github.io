FormBuilder with DataMate

A Powerful, Customizable Alternative to Google Forms

Key Features of FormBuilder

Advanced Field Types

Over 25 field types including Text, Dropdown, Multi-Select, Date, Time, Number, Checkbox, Radio, Textarea, Email, Star Ratings, Range Sliders, File Uploads, Conditional Fields, Calculated Fields, Signatures, Geolocation, Progress Bars, Captcha, Images, Videos, and more.

Dynamic Tables

Create interactive tables with dropdowns for item selection, price lookups from spreadsheet data, quantity inputs, and real-time totals (subtotal, tax, total), with the ability to add or remove rows dynamically.

DataMate Integration

Trigger custom data processing with seamless integration, copying form data across multiple sheets, updating inventory, and running complex workflows post-submission.

Flexible Submission Options

Choose between Single Cell or Table Row submissions, with dynamic table data serialized as JSON for precise control over data storage.

Dynamic Dropdowns

Populate dropdowns from spreadsheet ranges (e.g., =Sheet1!A:A) or custom lists, keeping options current with real-time updates.

Conditional Logic

Enhance user experience with smart forms where fields appear based on responses (e.g., show a text field if a checkbox is checked).

Custom Styling

Modern, responsive design with customizable containers, headers, and footers to match your brand’s look and feel.

File Management

Upload files directly to Google Drive with automatic link generation and customizable permissions for easy access.

Real-Time Calculations

Perform client-side calculations (e.g., =Quantity*2 or dynamic table totals) that update instantly as users input data.

Spreadsheet-Driven Data

Leverage spreadsheet ranges for static tables or dynamic table options, ensuring data consistency and flexibility.
